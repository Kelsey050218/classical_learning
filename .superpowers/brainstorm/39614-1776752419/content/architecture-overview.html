《经典常谈》伴学课件 - 产品架构总览

这是一个「读书 UP 主养成」为核心的 H5 伴学工具，请浏览整体架构

产品架构图 - 5 大模块 + 核心流程
🎯 用户入口
免登录访问 → 选填「姓名 + 班级」→ 生成个人学习 ID → 绑定所有数据
📱 首页
  • 学习进度总览
  • 项目规则说明
  • 原文速查悬浮
📚 学习中心
  • 3 大项目
  • 6 个子项目
  • 递进解锁机制
🛠️ 工具中心
  • AI 工具 (豆包)
  • 创作工具 (剪映/可画)
  • 预设 Prompt 模板
🏆 成果展厅
  • 作品卡片展示
  • 点赞/投票评比
  • 留言互动
👤 我的中心
  • 个人学习档案
  • UP 主素材库
  • 勋章墙/评价档案
💾 数据流向
学习任务 → 工具调用 → 成果提交 → 自动归集素材库 → 展厅展示 → 评价反馈

💡 核心设计思想：项目化学习驱动，学生以「成为读书 UP 主」为目标，通过完成递进式任务， 使用工具中心的生产力工具，创作内容并提交到成果展厅，最终形成个人作品集。
